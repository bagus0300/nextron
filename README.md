## Requirements

- `"node": ">=8.2.0"`
- `"npm": ">=5.2.0"`

## Install

```bash
$ npm install --global nextron
```

## Usage

```bash
$ nextron init <YOUR-APP-NAME>
$ cd <YOUR-APP-NAME>
$ yarn dev # for development
$ yarn build # for packaging the electron app
```
